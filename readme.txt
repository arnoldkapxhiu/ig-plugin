A standalone gallery plugin that allows users to upload and manage images below the WordPress editor.

== Description ==

Independent Gallery is a powerful plugin that allows you to create and manage galleries directly from the WordPress editor. Key features include:

- Drag-and-drop image reordering.
- Customizable columns and image sizes.
- Lightbox with zoom, slideshow, and fullscreen options.
- Easy-to-use shortcode for displaying galleries.

== Installation ==

1. Upload the `independent-gallery` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[ig_gallery]` shortcode to display galleries in your posts.

== Frequently Asked Questions ==

= How do I add a gallery? =

Edit a post, and you’ll see the "Gallery" meta box below the editor. Click "Add Images" to upload images and configure the gallery settings.

= Can I change the number of columns? =

Yes! You can set the number of columns (1–10) in the gallery settings.
